# templates/

suri-hr 角色的技能模板目录：存放角色创建时使用的标准模板文件。

## 模板文件

| 文件 | 用途 |
|------|------|
| `department_entry.md` | 部门录入模板 |
| `process_file.md` | 流程文件模板 |
| `role_files_map.md` | 角色文件权限地图模板 |
| `role_skills.md` | 角色技能模板 |
| `role_soul.md` | 角色 Soul 文件模板 |
| `rule_file.md` | 规则文件模板 |
