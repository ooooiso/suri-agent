# 监控运维

> 定义系统监控、角色监控、性能分析和故障排查机制。

---

## 运维全景

```
监控运维
├── 角色监控 — 角色健康度、活跃度、状态
├── 性能分析 — 任务执行时间、资源消耗
├── 日志策略 — 分类存储、日志级别
└── 故障排查 — 常见问题处理
```

---

## 策略总览

| 领域 | 维度 | 指标 |
|------|------|------|
| 角色 | 健康度 | 响就时间、错误率 |
| 角色 | 活跃度 | 任务完成量、学习频率 |
| 性能 | 任务 | 执行时间、资源消耗 |
| 日志 | 分类 | debug/info/warning/error |
| 故障 | 自愈 | 自动重试、升级路径 |
