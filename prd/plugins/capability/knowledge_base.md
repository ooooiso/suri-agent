# 知识库管理

> 定义跨角色共享知识的存储和管理机制。

---

## 一、知识库定位

```
记忆（角色级）       知识库（共享级）
  ├─ 角色经验          ├─ 最佳实践
  ├─ 角色偏好          ├─ 代码规范
  └─ 角色模式          ├─ 业务知识
                       └─ 学习成果
```

## 二、知识来源

1. **角色贡献** — 角色通过 role_learner 生成的知识片段
2. **用户输入** — 用户直接添加的知识
3. **文档同步** — doc_sync 从代码仓库同步的知识
4. **系统内置** — 系统预设的通用知识

## 三、存储

知识库存储在 `~/.suri/data/knowledge/` 目录下：

```yaml
~/.suri/data/knowledge/
├── best_practices/      # 最佳实践
├── code_standards/      # 代码规范
├── domain_knowledge/    # 业务知识
└── shared_patterns/     # 共享模式
