## 4. 关键测试说明

### 4.1 V3.0 综合测试（46 项）

覆盖：
- 核心角色内部标识格式 + 别名解析
- 非开发角色修改核心角色标识被拒绝
- 昵称显示/热重载/回退/播报
- 所有面向用户消息来源为 suri
- 单任务步骤展示 + 多任务进度看板
- 新需求创建独立 Agent + 多 Agent 状态独立
- 无依赖子任务并行 + Agent 上下文隔离
- 创建扩展部门 + 部门负责人二次分派
- 禁止删除核心角色 + 核心角色自动重建
- 非 dev 修改代码被拒绝 / dev 可修改代码 / review 只读
- 经验日志创建 + 标签过滤
- 终端状态卡片格式 + 精简版
- 消息总线持久化恢复 + 中断处理

### 4.2 1000 轮压力测试（48 项）

覆盖：
- 消息 ID 唯一性（UNIQUE constraint）
- 任务状态机流转（pending→in_progress→completed）
- 经验完整生命周期（保存/读取/触发计数/上下文注入）
- 边界情况（空消息/超长消息/Unicode/引号转义/JSON/XSS/SQL注入/控制字符）
- 500 轮生活化对话模拟 + 500 轮任务对话模拟
- 数据库完整性 + 高频并发模拟

### 4.3 100 种用户需求模拟（110 项）

覆盖 10 个类别：
1. 问候/闲聊（10 种，应无调度）
2. 开发维护（20 种，应匹配 suri_dev）
3. 人力资源（15 种，应匹配 suri_hr）
4. 审查审核（15 种，应匹配 suri_review）
5. 统计分析（15 种，应匹配 suri_stats）
6. 多角色混合（10 种）
7. 未知需求（10 种）
8. 边界/异常（5 种）
9. 部门匹配验证（5 种）
10. 技能匹配验证（5 种）
