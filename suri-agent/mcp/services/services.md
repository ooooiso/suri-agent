# services/

MCP 具体服务目录：存放各类可自增长的 MCP 扩展服务。

## 设计原则

- 每个服务独立目录，继承 `BaseMCPService`
- 服务由 `MCPRegistry` 动态加载，无需修改主程序
- 新增服务只需在此目录下创建新文件夹并注册

## 现有服务

| 服务 | 功能 | 状态 |
|------|------|------|
| `code_execution/` | 代码执行环境 | 预留 |
| `filesystem/` | 文件系统操作 | 预留 |
| `web_search/` | 网页搜索 | 预留 |
